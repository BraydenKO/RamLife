saveNotes method

Future<void> saveNotes (List<Note> notesList, List<int> readNotes)

Uploads the user's notes to the database.

This function saves the notes along with a record of the notes that were already read. Those notes will be deleted once they are irrelevant.

This should be re-written to only accept a list of JSON elements, as this creates a dependency between this library and the data library. This should also probably not persist the read notes in the database (ie, keep them local).

Implementation

static Future<void> saveNotes(List<Note> notesList, List<int> readNotes) async => _notes
	.document(await Auth.email)
	.setData({
		"notes": [
			for (final Note note in notesList)
				note.toJson()
		],
		"read": readNotes
	});
