AndroidNotification.reminder constructor

const AndroidNotification.reminder({bool root: false })

An optimal Android notification configuration for reminders.

If root is true, the notification is considered the group "summary", which is like a header for notifications.

Implementation

const AndroidNotification.reminder({bool root = false}) :
	importance = Importance.High,
	priority = Priority.High,
	style = AndroidNotificationType.normal,
	color = RamazColors.blue,
	channelId = "reminders",
	channelName = "Reminders",
	channelDescription = "When reminders are due.",
	groupId = "reminders",
	playSound = true,
	shouldVibrate = true,
	isGroupSummary = root,
	showChannelBadge = true,
	icon = null,
	styleInfo = null,
	showLight = true;
