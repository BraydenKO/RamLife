getReminders method

List<int> getReminders ({@required List<Reminder> reminders, @required Letters letter, @required String period, @required String subject })

Returns all reminders from a list of reminders that should be shown.

All possible parameters are required. This function delegates logic to ReminderTime.doesApply

Implementation

static List<int> getReminders({
	@required List<Reminder> reminders,
	@required Letters letter,
	@required String period,
	@required String subject,
}) => [
	for (int index = 0; index < reminders.length; index++)
		if (reminders [index].time.doesApply(
			letter: letter,
			period: period,
			subject: subject
		)) index
];
