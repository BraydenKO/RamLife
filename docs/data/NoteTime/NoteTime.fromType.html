NoteTime.fromType constructor

NoteTime.fromType({@required NoteTimeType type, @required Letters letter, @required String period, @required String name, @required bool repeats })

Instantiates new NoteTime with all possible parameters.

Used for cases where the caller doesn't care about the NoteTimeType, such as a UI notes builder.

Implementation

factory NoteTime.fromType({
	@required NoteTimeType type,
	@required Letters letter,
	@required String period,
	@required String name,
	@required bool repeats,
}) {
	switch (type) {
		case NoteTimeType.period: return PeriodNoteTime(
			period: period,
			letter: letter,
			repeats: repeats,
		); case NoteTimeType.subject: return SubjectNoteTime(
			name: name,
			repeats: repeats,
		); default: throw ArgumentError.notNull("type");
	}
}
