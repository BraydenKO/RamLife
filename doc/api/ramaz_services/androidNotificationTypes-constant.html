androidNotificationTypes top-level constant

const androidNotificationTypes = const {AndroidNotificationType.message : AndroidNotificationStyle.Messaging, AndroidNotificationType.image : AndroidNotificationStyle.BigPicture, AndroidNotificationType.normal : AndroidNotificationStyle.Default}

Maps the abstract AndroidNotificationType to AndroidNotificationStyles.

This is done so other libraries that import this module do not need to import flutter_local_notifications.

Implementation

> androidNotificationTypes = {
	AndroidNotificationType.message: AndroidNotificationStyle.Messaging,
	AndroidNotificationType.image: AndroidNotificationStyle.BigPicture,
	AndroidNotificationType.normal: AndroidNotificationStyle.Default,
}
