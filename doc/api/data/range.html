range function

Iterable<int> range (int stop)

A Python-like range function.

Like Python's range, this function returns an iterator that gives all values from 0 until stop, with 1 at a time. This is simply for iterating over lists while keeping the index.

Implementation

Iterable<int> range(int stop) sync* {
	for (int index = 0; index < stop; index++) {
		yield index;
	}
}
